# 🌐 SIPOC_00: End-to-End Logistics Overview
## نظرة شاملة على دورة حياة الشحنة

> هذا المخطط يمثل النظرة الشاملة (High-Level) لمشروع Green Box، حيث يوضح المسار الحرج للشحنة من لحظة نقرة العميل على "طلب" وحتى استلامها عند باب المنزل.

### 🔄 المخطط العام (The Master SIPOC)

| المورد (S) | المدخلات (I) | العملية (P) | المخرجات (O) | العميل (C) |
| :--- | :--- | :--- | :--- | :--- |
| العميل / المتجر | بيانات الطلب + العنوان | **1. إنشاء الطلب** | طلب مؤكد رقمياً | قسم العمليات |
| المستودع | الشحنة + الصناديق | **2. التجهيز والفرز** | شحنة جاهزة للتوزيع | فريق التوصيل |
| نظام التوجيه | قائمة العناوين | **3. جدولة المسارات** | خطة سير الرحلة | السائقين |
| السائقين | المركبة + الشحنة | **4. رحلة التوصيل** | شحنة مسلّمة | العميل النهائي |
| إدارة الأداء | بيانات التسليم | **5. تحديث الحالة** | تقرير أداء (KPIs) | الإدارة المالية |

### 🎯 الحدود الجغرافية والزمنية (Process Boundaries)
* **نقطة البداية:** استلام طلب العميل في النظام.
* **نقطة النهاية:** تحديث حالة الطلب إلى "تم التسليم" (Delivered) واستلام التغذية الراجعة.
