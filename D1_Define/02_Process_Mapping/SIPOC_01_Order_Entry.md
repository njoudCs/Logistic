# 📥 SIPOC_01: Order Entry & Validation
## استقبال ومعالجة الطلبات

| المورد (S) | المدخلات (I) | العملية (P) | المخرجات (O) | العميل (C) |
| :--- | :--- | :--- | :--- | :--- |
| العميل (App/Web) | الاسم + رقم الجوال | **استلام الطلب** | مسودة طلب | نظام الـ CRM |
| خرائط جوجل / API | إحداثيات الموقع | **تأكيد العنوان** | عنوان دقيق (Geo-tagged) | المستودع |
| العميل | تأكيد الدفع | **معالجة الدفع** | إيصال دفع | قسم المالية |

**⚠️ مخاطر هذه المرحلة:** إدخال عنوان غير دقيق هو المسبب الرئيسي لـ 30% من تأخيرات الميل الأخير.
