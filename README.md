# Logistic
# 🚚 Logistic Support System (LSS) (Green Box)

## 📌 Overview

This repository is used to **manage and track Green Box project**.
It serves as a centralized workspace for organizing project tasks, monitoring progress, and documenting logistics operations.

> ⚠️ This is **not a coding project**. Instead, GitHub is used as a **project management and tracking tool**.

---

## 🎯 Project Purpose

The goal of this project is to:

* Track logistics operations and activities
* Monitor delivery performance and delays
* Organize tasks and responsibilities
* Ensure smooth communication and workflow management

---

## 📋 How This Repository is Used

This repository supports the project through:

### 🧩 Task Management

* Issues are used to track tasks and activities
* Each task includes description, status, and ownership

### 📊 Project Tracking

* GitHub Projects (boards) are used to:

  * Track progress (To Do / In Progress / Done)
  * Monitor workload distribution
  * Follow up on delayed tasks

### 📁 Documentation

* Store project-related files (reports, Excel sheets, insights)
* Maintain structured documentation for reference

---

## 📈 Key Focus Areas

* Delivery tracking
* Delay analysis
* Driver performance monitoring
* Operational efficiency
